# SELT Project 2025

## GiftWiser - Smart Gift Planning and Event Management Application

**GiftWiser is a simple web application that allows users to manage events, store recipient details, and organize personalized gift ideas. The purpose of this project is to prevent duplicate gifting and make event planning easier and more organized.**

### Getting Started 

**Clone the Repository**

git clone https://github.com/hjmjohnsonSELT2025/giftwiseproject-selt_2025_team_07.git

cd giftwiseproject-selt_2025_team_07

**Install Dependencies**
    
    bundle install

**Set Up the Database**
    
    rails db:create
    rails db:migrate
    rails db:seed   # optional

**Tailwind CSS Setup**

If you encounter a missing stylesheet or asset pipeline error, run:

    rails tailwindcss:install

**Running the Application**
    
    bin/dev
        or
    rails server
